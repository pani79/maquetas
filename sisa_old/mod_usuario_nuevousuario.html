Seguridad

La cuenta del usuario puede tener diferentes niveles de seguridad. Seleccionar la configuración de seguridad apropiada.

Complejidad de contraseña

La complejidad de la contraseña define el nivel de complejidad que el sistema requerirá para establecer la contraseña de esta cuenta de usuario. Cuanto mayor es la complejidad de contraseña menor es la probabilidad de que otra persona pueda descifrarla. Para mayor información consulte la ayuda en línea.

El vencimiento de la contraseña sirve para establecer si la contraseña de esta cuenta de usuario expira, de esta manera se fuerza al usuario a que la cambie periódicamente. Si se configura un período y el usuario no cambia su contraseña antes del vencimiento, entonces la cuenta de usuario se deshabilita y debe ser rehabilitada. Horario de trabajo

El horario de trabajo sirve para limitar en que días y horarios el usuario puede realizar transacciones en el sistema (gestión de datos en los registros a los que tenga acceso). Si se utiliza esta caracterìstica y el usuario ingresa al sistema fuera del horario de trabajo, sólamente podrá consultar información pero no podrá modificarla o alterarla.
